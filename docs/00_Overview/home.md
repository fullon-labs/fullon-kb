# FullOn Home

## What FullOn is
FullOn Network is a decentralized, scalable, low-latency layer-1 blockchain with fast finality and multi-chain interoperability.

This knowledge base helps answer questions about:
- what FullOn is
- how the protocol works
- what each ecosystem and DApp does
- how users should start
- how operators and developers interact with the network
